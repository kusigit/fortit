# `<fwc-card>`  

## Example usage

### Standard  

```html
<fwc-card>
    <div>Some content</div>
</fwc-card>
```

### With padding

```html
<fwc-card padded>
    <div>Some content</div>
</fwc-card>
```

### Properties/Attributes

| Name            | Type      | Description
| --------------- | --------- |------------
| `padded`        | `boolean` | Add padding to card content.
